#  ATM Simulation (Caixa Eletrônico)

A simple **ATM interface simulation** developed with **HTML, CSS, and JavaScript**.  
This project was created as part of programming exercises at **Etec**, demonstrating how to simulate basic banking operations in a web environment.

---

##  Features
- **Login screen** for account access.  
- **Balance inquiry** to check available funds.  
- **Deposit operation** to add money.  
- **Withdrawal operation** with validation (e.g., sufficient balance).  
- **Transaction history** displayed dynamically.  
- Extra pages: **About** and **Contact**.  
- Responsive and user‑friendly design.  

---

##  Technologies Used
- **HTML5** → Page structure.  
- **CSS3** → Styling and layout.  
- **JavaScript (ES6)** → Logic for ATM operations and DOM manipulation.

##  How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/GabrielSantana1996sp/Caixa-Eletronico.git
   cd Caixa-Eletronico
   ```
2. Open `index.html` in your browser.  
3. Navigate through the **ATM simulation**, **About**, and **Contact** pages.  
4. Perform operations like **deposit**, **withdraw**, and **check balance**.  

---

##  Purpose
The goal of this project is to:
- Practice **front‑end development** with HTML, CSS, and JavaScript.  
- Simulate a **real‑world ATM workflow** in a browser environment.  
- Strengthen understanding of **DOM manipulation, event handling, and page navigation**.
